# Dashboard Ciberbullying — Brasil 2020–2026

**Projeto de Análise de Dados (PAD)**

Análise exploratória de denúncias de cyberbullying no Brasil com base em dados públicos (SaferNet, Anuário Brasileiro de Segurança Pública e Ministério Público).

## Objetivo
Visualizar a evolução temporal, distribuição por plataforma, região e perfil das vítimas de violência online.

## Estrutura do Projeto
- `index.html` → Dashboard interativo (GitHub Pages)
- `dados/` → Arquivos CSV com os dados
- `scripts/py.py` → Script de processamento dos dados

## Como rodar localmente
1. Clone o repositório
2. Abra o `index.html` no navegador
3. (Opcional) Execute `py.py` para gerar/atualizar dados
